# Mirror for Photoshop Plugin

Photoshop UXP plugin for Mirror for Photoshop - real-time preview on iOS.

## Installation

1. Download the latest `.ccx` file from [Releases](https://github.com/hex/ps-mirror-plugin/releases)
2. Double-click the file to install in Photoshop
3. Find Mirror for Photoshop in Window > Extensions

## Requirements

- Adobe Photoshop 2021 (v22.0) or later
- Mirror for Photoshop iOS app
- Mirror server running on your computer
