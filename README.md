# PS Mirror Plugin

Photoshop UXP plugin for PS Mirror - real-time preview on iOS.

## Installation

1. Download the latest `.ccx` file from [Releases](https://github.com/hex/ps-mirror-plugin/releases)
2. Double-click the file to install in Photoshop
3. Find PS Mirror in Window > Extensions

## Requirements

- Adobe Photoshop 2021 (v22.0) or later
- PS Mirror iOS app
- PS Mirror Server running on your computer
